# ПОГРАНИЧЬЕ

Кооперативная топ-даун **action RPG** в живом открытом мире (в духе RimWorld)
с элементами выживания и adventure-режима Dwarf Fortress: классы и таланты,
тактический бой, живые фракции, тёмные пути и путь к божественности.
Работает в браузере, кооп по сети до 4 игроков.

## Запуск

```bash
npm install
npm start
```

Открой `http://localhost:8080`. Друзья заходят по адресу из консоли сервера
(`http://<твой-IP>:8080` — по локальной сети или интернету при открытом порте).

Пересобрать графику (скачивает CC0-паки Kenney, недостающее генерирует само):

```bash
npm run assets
```

Тесты (юнит + сетевой смоук с двумя ботами):

```bash
npm test
```

Свой мир: `SEED=12345 npm start` (Windows PowerShell: `$env:SEED=12345; npm start`).
Мир автосохраняется в `saves/world.json` раз в минуту и при Ctrl+C
(папку можно перенести переменной `SAVES_DIR`).

## Панель управления сервером

По адресу `/admin` живёт панель хозяина сервера. Ключ доступа печатается
в консоли при запуске (ссылка сразу с ключом); постоянный ключ задаётся
переменной `ADMIN_KEY`. Без ключа API отвечает 403 — не свети ссылку игрокам.

Что умеет:

- **Статус**: день/время, погода, событие, мощь Тьмы, арена, нагрузка
  тика, таблицы игроков и поселений — обновляется само.
- **Гейм-мастер**: исцелить/телепортировать/кикнуть игрока, выдать
  предмет или монеты, сменить время и погоду, наслать мировое событие
  (кровавую луну — по кнопке), объявление всем. Все вмешательства
  пишутся в летопись как деяния «Руки судьбы».
- **Сейвы**: сохранить сейчас, **скачать бэкап**, **загрузить сейв на
  лету** — мир и профили игроков заменяются без рестарта (сид должен
  совпадать). Это спасение для бесплатного Render: диск там стирается
  при рестарте — скачал бэкап вечером, залил обратно после рестарта,
  и мир продолжился.

## Деплой в интернет (бесплатно, Render)

В репозитории лежит [render.yaml](render.yaml) — готовый blueprint:

1. Зарегистрируйся на [render.com](https://render.com) (можно через GitHub).
2. Dashboard → **New → Blueprint** → выбери этот репозиторий → **Apply**.
3. Через пару минут игра доступна по адресу вида
   `https://pogranichye.onrender.com` — кидай ссылку друзьям.

Особенности бесплатного тарифа:

- **Сейвы недолговечны**: диск у free-инстанса эфемерный — при каждом
  редеплое или рестарте `saves/world.json` стирается и мир начинается
  заново. Для постоянного мира нужен платный диск (Render Disk) или
  свой VPS.
- **Засыпание**: без посетителей ~15 минут сервис засыпает, первый
  вход после этого занимает ~30-60 секунд (и тоже сбрасывает мир).
- Порт задаётся переменной `PORT` автоматически, WebSocket работает
  из коробки. Сид мира — переменная окружения `SEED`.

Альтернатива без аккаунтов: пробросить свой порт 8080 наружу
(`ngrok http 8080` или туннель Cloudflare) — тогда мир живёт у тебя.

## Управление

| Клавиша | Действие |
|---|---|
| WASD | движение |
| ЛКМ | атака (удар / выстрел) |
| ПКМ | предмет левой руки: блок щитом / призыв элементаля / барьер / сеть |
| Пробел | перекат (неуязвимость в начале) |
| Q / X / R | активные способности класса (открываются на 2, 5 и 9 уровнях) |
| N | музыка вкл/выкл |
| E | действие: диалог, вход в данж, сундук, костёр, поднять союзника |
| T | перезарядка |
| 1–4 | смена оружия |
| Tab | инвентарь и экипировка (кукла со слотами) |
| C | лист персонажа: характеристики и таланты |
| J | журнал заданий (до 3 одновременно) |
| B | бестиарий: твари по тирам и где их искать |
| K | книга способностей: описания и раскладка Q/X/R |
| P | дипломатия: фракции, репутация, мощь Армии Тьмы |
| L | летопись: вести мира и события (не мельтешат на экране) |
| M | карта мира |
| F3 | отладка (пинг, ресимы, fps) |

## Что в мире

- **Сюжетные цепочки с выбором (9)**: отшельник Радогост, капитан
  Ярослава, странница Мирослава, кузнец-мастер Творимир (Молот гор —
  себе или дар деревне), вдова Милица, Голос из колодца — и три новых
  в дальних деревнях: **«Хворь»** (знахарь Богумил лечит мор, который
  сам устроил — разоблачи и сразись с Отравителем или возьми долю
  и молчи), **«Пропавший караванщик»** (иди по следам мужа Весняны;
  жив он в плену или погиб — решает сид мира; правда или ложь за
  200 монет), **«Голос болот»** (подношение чёрному идолу будит
  аватара: срази его — или заключи кровавый пакт: −1 сердце навсегда,
  +10% урона навсегда).
- **ВЫЖЖЕННЫЕ ЗЕМЛИ**: за обсидиановым порталом (настройка печати —
  10 кристаллов, единожды) лежит вулканический регион: пепельные пустоши,
  лавовые реки (перекатом можно проскочить!), обсидиановые гряды и жилы
  тлеющего кристалла. Шесть огненных тварей тира 4-5: саламандры,
  пепельные гончие, взрывающиеся элементали, ифриты-призыватели,
  магмовые големы с раскалённой коркой и лавовые змеи. В лагере
  огнеходцев — торговец Жарох, зачарователь Искра и Огневзор с тремя
  испытаниями огня (награда — доспех ордена). На севере — логово
  Старшего голема, сундук с реликвией и трон ВЛАДЫКИ ПЕПЛА: срази
  голема, принеси 15 кристаллов — и ритуал разбудит финального босса
  региона. Существо огня: пламя ему смешно (×0.4), лёд — погибель
  (×1.4, реакции ПАР/РАСКОЛ решают). Три фазы: лавовые кулаки, ифриты
  из завесы, в ярости — двойные пепельные лавины. Награда — легендарный
  молот «Сердце вулкана» (раны кипят лавой), сет ордена, реликвия;
  имя первого победителя мир запоминает навсегда.
- **Сеты экипировки**: четыре комплекта с бонусами за 2 и 4 части —
  «Волчья стая» (скорость, кровотечение), «Ледяной чертог» (мана,
  заморозка магией), «Ночная тень» (уворот, удары в спину +40%),
  «Пепельный орден» (сердца, пепельный шаг по лаве, огненная нова).
  Части падают с боссов, чемпионов арены и элиты Выжженных земель.
- **Реликвии**: шесть эпиков с уникальными свойствами — «Гнев небес»
  (молния сама бьёт врагов), «Перо феникса» (спасает от смерти),
  «Кольцо инея» (перекат — ледяная нова), «Жажда крови» (заряды за
  убийства), «Шипастый панцирь», «Сапоги ветра». Зачарователь перекуёт
  любой эпик в реликвию за 6 кристаллов и 120 монет.
- **Арена-колизей**: распорядитель игр Боривой в первой деревне ждёт
  смельчаков — взнос 25 монет, волны врагов всё злее, каждая пятая —
  чемпион. Награда растёт с волной, рекорд сервера хранится вечно.
- **Задание дня**: каждое утро старейшины объявляют общую цель (истребить
  тварей или принести ресурсы) — монеты, опыт и репутация раз в день.
- **Ранги фракций**: репутация 25 — Друг, 50 — Защитник (дар 100 монет),
  80 — Герой (эпический дар фракции и слава на весь мир).
- **Классовые ресурсы**: у мага — чародейские заряды, у воина — ЯРОСТЬ
  (растёт от ударов и боли, сжигается способностями: до +66% силы),
  у вора — КОМБО (серия попаданий до 5 очков, +12% за очко к приёмам),
  у жреца — БЛАГОДАТЬ (лечение союзников заряжает Кару и Освящение).
- **Реакции стихий**: лёд, огонь и яд взаимодействуют — огонь по
  замороженному = ПАР (взрыв по области), удар мили по льду = РАСКОЛ
  (+35% урона), огонь по отравленному = ТОКСИН (вспышка и яд летит на
  соседей). Кооп-синергии: маг морозит — воин раскалывает.
- **Живой ИИ врагов**: стаи окружают с флангов, а не толпятся в линию;
  раненые звери и люди дрогнут и отступают (нежить и Тьма — никогда);
  удар по одному будит его сородичей рядом; ловкие твари дёргаются
  вбок — попробуй попади издалека.
- **Тактический бой**: щитоносцы блокируют удары спереди (фланг или стан!),
  оркские жрецы лечат стаю, демонологи и некроманты призывают миньонов,
  споры взрываются при смерти, боссы бьют по области с красным телеграфом.
- **Приключения**: охота на именных зверей (мировое событие с трофеями),
  курганы с рунной загадкой (I→II→III→IV — ошибёшься, встанут стражи),
  8 тайников-сундуков в глуши.
- **Тёмный путь**: грабь караваны (груз твой, но репутация и розыск 💀 —
  за головой придут охотники; виру платят старейшинам) и подписывай
  кровавые контракты у жреца: 8 минут испытания (элитный мир / стеклянная
  пушка / орда) за +75% монет, +50% опыта и сундук в финале.
- **Мана — настоящий ресурс**: синяя полоска у каждого класса (воин 20,
  вор 25, маг 40, жрец 45 + 4 за очко ИНТ). В бою мана течёт тонкой
  струйкой; через 5 с без каста начинается «медитация» — реген
  разгоняется до ×4 за ~11 секунд отдыха, а у костра ещё в полтора
  раза щедрее. Посохи не перезаряжаются — каждый каст пьёт ману (1-4).
  Механики мага: Чародейские заряды (агрессивный ритм = до +20% урона
  магии, пипсы в HUD), Мана-всплеск (+3 маны за убийство магией) и
  Кровавый каст (нет маны — плати сердцем: 1 сердце = 10 маны).
- **Вознесение (уровень 20)**: на пределе смертного пути Древний обелиск
  предлагает Ритуал Вознесения — 20 кристаллов, 10 металла, 500 монет и
  три волны древних стражей (последний — Каменный король). Погибнешь —
  дары сгорят. Выстоишь — станешь БОГОМ: +4 ко всем атрибутам, +3 сердца,
  +8% скорости, реген, способности на четверть быстрее, золотой нимб.
- **Средневековая рубка**: девять ближнебойных врагов — гоблины,
  хобгоблины, гноллы, орки-рубаки и орки-рыцари, упыри, огры, минотавры
  и орки-вожди; арсенал пополнили булава, кистень, моргенштерн,
  двуручный меч и алебарда.
- **Мировые события**: кровавая луна (элита повсюду), разломы с
  демонами, метеориты с кристаллами, странствующие торговцы; после
  финала войны события чаще, а узурпатору Сердца мстит культ Тьмы.
- **Данжи 2.0**: ловушки-лезвия (перекат проскакивает), дверь босса
  заперта — ключ у мини-босса (орк-вождь или минотавр), лестница на
  нижний этаж (сложнее и богаче), проклятые данжи с элитой и щедрым лутом.
- **Свой дом**: покупается у старейшины (150 мон., репутация 20) —
  кровать-респаун с полным сном, личный сундук.
- **Онбординг**: цепочка первых целей в HUD ведёт новичка от знакомства
  со старейшиной до первой покупки.
- **Война с Тьмой (эндгейм)**: кампания у старейшин — союз трёх фракций
  (репутация 25), сбор реликвий (кристаллы, Сердце Тени с Хранителя,
  Древний осколок с боссов), великий ритуал ослабляет врата Цитадели,
  штурм гарнизона и финальный выбор у Сердца Тьмы: уничтожить навсегда
  или подчинить. Награда — ЛЕГЕНДАРНОЕ оружие по классу: Клинок рассвета
  (лечит за убийства), Посох зари (тройная цепная молния), Ветер степей
  (двойной выстрел); за тёмный путь — ещё и Печать Тьмы.
- **Владыки — тир 6**: все боссы вынесены в высший тир бестиария и
  дерутся по-новому. Рывки через арену с красной телеграф-линией
  (уйди с траектории!), свита на смене фаз (гоблины огра, големы
  короля, волчий вой вожака, живые бомбы ведьмы), ЯРОСТЬ на последней
  трети здоровья — босс быстрее и бьёт чаще. Каменный король закрыт
  гранитной коркой спереди, Болотная колдунья лечит свиту, Хранитель
  сердца без конца плодит бесов. Элитные монстры с аффиксами (Свирепый,
  Живучий, Стремительный, Золотой) отмечены звездой — вдвое больше
  опыта и щедрый дроп.
- **Кузнец-эндгейм**: перековка вещей в высшую редкость (металл, для
  украшений — кристаллы), разборка ненужного на материалы.
- **Кооп-глубина**: общий сундук отряда в таверне, передача вещей
  союзнику из инвентаря, совместная молитва у святилища (бафф группе).
- **Атмосфера**: генеративная фоновая музыка (день/ночь/подземелья),
  дожди весной и осенью, снегопады зимой.
- **Способности классов (Q/X/R) и Книга способностей (K)**: у каждого
  класса ВОСЕМЬ активных умений лесенкой 2→16 уровня (новые — уже на
  3, 6 и 8!); какие три висят на клавишах — решаешь сам в Книге (K).
  Воин: Мощный удар, Удар щитом (стан в упор), Боевой клич, Второе
  дыхание, Вызов (танкует агро!), Вихрь, Героический рывок,
  Несокрушимость. Маг: Огненная волна, Ледяное копьё (метка льда),
  Ледяная нова, Ледяная броня, Возгорание (детонация дотов), Телепорт,
  Метеор, Живая бомба. Вор: Рывок теней, Ослепляющий порошок,
  Дымовая завеса, Ядовитый клинок, Уклонение (+40% уворота), Град
  клинков, Шаг сквозь тень, Ковёр шипов. Жрец: Волна света, Свет
  прикосновения, Кара небес, Сияние, Епитимья (луч: кара врагу или
  лечение своему), Щит веры, Освящение, Дух-хранитель.
- **Армия Тьмы**: Чернокаменная Цитадель на юге мира. Мощь Тьмы растёт
  с каждым днём, войска (солдаты, стрелки, чернокнижники и рыцари Тьмы)
  маршируют на добрые деревни; павшие поселения становятся фортами Тьмы
  и кормят её рост. В Цитадели ждёт Лорд Тьмы — разгром гарнизона
  вдвое подрубает мощь, освобождение фортов тоже ослабляет Тьму.
- **Экипировка по слотам**: голова, грудь, ноги, левая рука (щит —
  блок на ПКМ), правая рука (оружие), два аксессуара и кольцо. Красивая
  карточка предмета с редкостью, статами и ценой — при наведении.
- **Фэнтези-бой**: ближний бой (меч, секира) с ударом по дуге + дальний
  (луки, арбалет, метательные ножи, огненные бомбы) + магия: посохи огня
  и льда, огненные шары со взрывом по области, цепная молния. Перекат с
  i-frames, тряска экрана, паттерны магии врагов, боссы с фазами.
- **Разрушение построек**: у стен, башен, деревьев и валунов есть
  прочность — меч крошит по блоку, секира быстрее, огненные шары и бомбы
  сносят по несколько блоков разом, горящие стрелы (расходник) дают лукам
  разрушающую силу. На месте стен остаются обломки, из валунов выпадает
  металл. Вандализм в деревнях карается репутацией и стражей.
- **Ритуалы призыва**: святилища при нехватке защитников призывают
  духа-хранителя из иного мира (15% шанс — ритуал выходит из-под контроля
  и в мир вырываются демоны); захваченные бандитами деревни проводят
  тёмные ритуалы, порождая стаи демонов и бесов.
- **Оружие — предметы**: 4 ячейки в руках (клавиши 1–4), остальное в сумке.
  Покупается у торговцев, продаётся им же (кнопка «Продать вещи»), падает
  из сундуков и боссов.
- **Открытый мир 768×768**: биомы из шума (вода, леса, пустыни, болота, скалы),
  9 поселений трёх фракций, дороги, 20 подземелий и лагерей разбойников,
  утроенные достопримечательности: 3 кургана, 3 обелиска, 3 источника,
  3 каменных круга и 14 тайников в глуши.
- **Живые цивилизации и экономика**: деревни добывают четыре ресурса
  (пшеница с полей по сезонам, древесина из лесов, металл из шахт у скал,
  кристаллы у болот) и тратят их на строительство (дома, поля, шахты,
  святилища, башни — появляются на карте на глазах!), армию (стража
  требует металл) и магические ритуалы (урожай, оберег от осад). Караваны
  возят излишки туда, где дефицит — и их грабят стаи. Деревни растут при
  сытости, вымирают в голод (остаются руины ☠), а ослабшие захватываются
  бандитами (⚔ на карте) — игрок может перебить захватчиков и освободить
  деревню (+опыт, +репутация). Богатые соседи враждуют за угодья, а игрок
  с репутацией может выступить миротворцем. Сезоны: летом урожай щедрее,
  зимой монстры смелее.
- **Живой мир**: у жителей распорядок дня (сон → работа → костёр), монстры
  нападают на деревни по-настоящему — гражданские разбегаются, стража
  встаёт стеной; вдали от игроков мир живёт абстрактно — стаи кочуют,
  караваны ходят между городами, случаются набеги. Всё пишется в журнал
  событий — его пересказывают в тавернах как слухи.
- **NPC с именами и памятью**: старейшины, торговцы, кузнецы (ковка
  оружия до +3), жрецы (исцеление, благословение), трактирщики (отдых,
  наём бойца-компаньона), охотники (заказы на шкуры) — помнят знакомых.
  Доска заказов гильдии у таверны — до 4 заданий на выбор.
- **Каждое здание полезно**: кровать — сон (+сердце), колодец — вода,
  шахта — добыть металл, святилище — благословение за ману, башня —
  осмотреться (метки логов на карте), поле — собрать колосья (если
  репутация позволяет), наковальня — кузнечный крафт (болты, бомбы,
  горящие стрелы).
- **Фракции и репутация**: цены, квесты и враждебность стражи зависят от
  твоих поступков. Убийство жителя не прощают.
- **ЖРЕЦ — четвёртый класс**: свет лечит своих и карает чужих. Посох
  света: снаряд, попавший в союзника, ЛЕЧИТ его (дружественный огонь
  впервые работает на тебя). Способности: Волна света (лечение отряда),
  Кара небес (столб света со станом), Щит веры (барьер всей группе).
  Три ветки талантов: Свет (аура, усиленное лечение и воскрешение
  павшего волной — капстоун «Длань Света»), Кара (урон света, поджог,
  +1 снаряд), Оплот (сердца, стена маны, несокрушимость).
- **Классы и прокачка**: Воин, Маг, Вор — свои стартовые наборы, спрайты и
  деревья из 9 талантов в 3 ярусах (вихрь стали, таран, двойной сгусток,
  заморозка, тень, криты ×3…). Уровни за опыт: +1 очко характеристики и
  +1 талант за уровень. Сила — урон в ближнем бою и +½ сердца за очко;
  Ловкость — скорость атаки и 2% уворота за очко; Интеллект — реген маны
  и урон магии; Удача — шанс, количество и редкость добычи. Криты,
  летящие цифры урона, опыт делится на всех в кооперативе.
- **Редкость предметов**: обычные, редкие (синие, ×1.6 статов) и
  эпические (фиолетовые, ×2.3); 14 видов оружия и 27 предметов
  экипировки, включая броню с бонусами атрибутов (доспех берсерка +2 СИЛ,
  корона предводителя +1 ко всем). Редкость дропа растёт от удачи, боссы дают минимум редкое.
  Легендарные вещи задуманы под сложнейшие задания и ритуалы.
- **РПГ-экипировка**: 4 слота (броня, голова, амулет, щит) — доспехи дают
  сердца, амулеты — скорость/урон/перекат; зелья лечения, маны и прыти.
  Экипировка продаётся у торговцев, падает из сундуков, элиты и боссов.
- **Выживание-лайт**: голод, еда, крафт у костра (бинты, стрелы, готовка),
  день/ночь с освещением от костров.
- **Данжи как в Gungeon**: комнаты запечатываются до зачистки, сокровищницы
  с оружием, комнаты боссов.
- **Кооп**: общий мир, поднятие павшего союзника (E), общие квесты.

## Устройство

- `server/` — авторитетная симуляция 30 Гц (Node.js + ws).
- `client/` — Canvas-рендер 480×270 с целочисленным пиксель-скейлом,
  предсказание своего игрока, интерполяция чужих, пули как события.
- `shared/` — общий код симуляции (движение, перекат, пули, паттерны),
  одинаково исполняется на клиенте и сервере.
- `tools/` — пайплайн ассетов: скачивание CC0-паков Kenney + процедурная
  генерация недостающих спрайтов в палитре DB32, сборка атласа.

Графика: [Kenney](https://kenney.nl) Tiny Dungeon / Tiny Town / Pixel Shmup /
Roguelike-паки (CC0) + [Dungeon Crawl Stone Soup tiles](https://opengameart.org/content/dungeon-crawl-32x32-tiles)
(CC0) для предметов и оружия + процедурные спрайты. Звук синтезируется на лету (WebAudio).
